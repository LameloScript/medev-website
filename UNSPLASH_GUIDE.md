# Guide des Images Unsplash pour les Projets

Ce document contient les mots-clés Unsplash recommandés pour trouver des images adaptées à chaque projet. Visitez [unsplash.com](https://unsplash.com) et recherchez ces mots-clés.

## Instructions
1. Allez sur [unsplash.com](https://unsplash.com)
2. Recherchez les mots-clés suggérés pour chaque projet
3. Téléchargez l'image en haute résolution
4. Renommez le fichier selon le nom indiqué
5. Placez l'image dans `/public/assets/projects/`

## Notes importantes

### Conseils pour le choix des images
- ✅ Privilégiez des images en orientation **paysage** (landscape) pour une meilleure intégration
- ✅ Choisissez des images avec une **haute résolution** (minimum 1920x1080px)
- ✅ Préférez des photos avec **des couleurs vives** et un bon contraste
- ✅ Évitez les images trop chargées ou avec trop de texte
- ✅ Assurez-vous que l'image représente bien le **concept du projet**

### Format et optimisation
- Format recommandé: **JPG** (pour réduire la taille du fichier)
- Taille maximale: **500 KB** (optimisez avec TinyPNG.com si nécessaire)
- Dimensions recommandées: **1920x1080px** ou **1600x900px**

### Attribution Unsplash
Bien qu'Unsplash permette l'utilisation gratuite, il est recommandé de:
- Créditer le photographe dans votre code (commentaire HTML)
- Télécharger uniquement pour un usage commercial si votre site est commercial

## Mise à jour rapide

Une fois les images téléchargées et placées dans le dossier, le site affichera automatiquement les nouveaux projets avec leurs images respectives dans la section "Nos Réalisations".

Pour vérifier:
1. Naviguez vers `/fr/projects` sur votre site
2. Les 11 projets devraient s'afficher avec leurs catégories
3. Utilisez le filtre pour trier par catégorie
